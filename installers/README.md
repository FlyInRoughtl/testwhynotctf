# installers/

This folder will contain USB/VM installer scripts for Windows and Linux.

Planned structure:
- windows/ (PowerShell/GUI)
- linux/ (bash/CLI)

See SPEC.md for the installer wizard requirements.

Quick bootstrap:
- Windows: windows/bootstrap.cmd
- Linux: linux/bootstrap.sh

TUI wizard:
- Windows: windows/wizard.cmd
- Linux: linux/wizard.sh
